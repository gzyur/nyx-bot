## Sobre

Nyx é a assistente oficial do meu servidor privado. Desenvolvida em Discord.js v14, conta com personalidade própria, sistema de níveis, moderação completa e integração com inteligência artificial para respostar personalizadas.

## Comandos

### 💬 Geral

| Comando | Descrição |
|---|---|
| `/ping` | Verifica se a Nyx está online e exibe a latência. |
| `/servidor` | Exibe informações detalhadas sobre o servidor. |
| `/comandos` | Lista todos os comandos disponíveis. |

### ⭐ Níveis

| Comando | Descrição |
|---|---|
| `/nivel [@membro]` | Exibe o perfil de nível de um membro. |
| `/ranking` | Exibe o top 10 de membros mais ativos do servidor. |

### 🛡️ Moderação

> Requer o cargo de moderador.

| Comando | Descrição |
|---|---|
| `/avisar @membro motivo` | Aplica um aviso formal ao membro. Ao terceiro aviso, o membro é notificado que será expulso. |
| `/expulsar @membro [motivo]` | Expulsa um membro do servidor com confirmação. |
| `/banir @membro [motivo] [dias]` | Bane um membro permanentemente com confirmação. |
| `/limpar quantidade [@membro]` | Apaga mensagens em massa no canal. |

### 🤖 Inteligência Artificial

A Nyx responde quando mencionada diretamente em canais autorizados.

```
@Nyx qual é o seu propósito aqui?
```
